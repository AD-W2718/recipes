## EN — Introduction

This project is a learning-oriented recipe website consisting of several pages: a homepage listing the names of all recipes, and a separate page for each individual recipe. It was built to practice the fundamentals of plain HTML markup. Throughout the project, I focused on three core elements:

- **Working with text** — structuring content using headings, paragraphs, and lists (e.g., ingredient lists or cooking steps).
- **Working with images** — adding photos of dishes, setting `alt`, `width`/`height` attributes, and placing images correctly within the content.
- **Working with links** — using both **relative** links (navigation from the homepage to individual recipe pages and back) and **absolute** links (links to external resources, sources, or social media).

Project structure:
- `index.html` — the homepage with the site title and a list of all recipe names;
- a separate HTML page for each recipe  with a full description, an image, and cooking steps.

By the end of this project, I will be able to confidently:
- build a semantically correct HTML structure for a multi-page site;
- properly insert and style images;
- distinguish between and correctly apply relative and absolute paths in links for navigation between pages;
- combine these elements into a cohesive, interconnected set of pages.
